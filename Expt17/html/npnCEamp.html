NPN Transistor Amplifier, CE configuration

The schematic is wired as shown in the diagram below. It is very similar to the schematic used for drawing the output characteristics. The AC signal is connected to the base, through a capacitor so that the biasing is not affected. Since the gain is high, we need a small input signal, smaller than the 80 mV available from WG. A divider network using 1k and 10k gives  7 mV output and that is fed to the base. This input signal is monitored on A3. The 7 mV signal is difficult to view even in the .5 volt scale. So a gain resistor Rg=1k is used for amplifying the input to A3 so that can be viewed better.
