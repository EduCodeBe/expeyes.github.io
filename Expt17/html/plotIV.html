Plotting of Current vs Voltage, for resistors.

Two resistances are connected as shown in the diagram. R2 (generally 1k) is used for measuring the current indirectly, ie from the voltage drop across it. The voltage at PV1 can be set using a slider. The displayed voltage is the voltage across R1. The current also will be displayed. The voltage can be varied within selected values and IV plots can be generated.

   

The screen shot shows the graphs for a 2.2k and 10k resistors along with a PN junction. The resistor is a linear element but the PN junction is non-linear, as evident from the plot.
